# Velozzy

Velozzy is a high-performance, custom-built web browser designed for speed, simplicity, and security. Developed by @keyzzilla and published by Keyzzilla Labs, Velozzy leverages a custom Chromium engine to provide a seamless browsing experience.
Features

    Custom Chromium Engine: Powered by a tailored engine for high-performance browsing.

    Smooth Navigation: Optimized for a fast and responsive user experience.

    Secure Password Vault: Integrated local storage for managing credentials securely.

    Extension Side-Loading Support: Full support for adding custom tools and extensions.

# Installation

**To install Velozzy on your system:**

    Run VelozzySetup.exe: Locate the installer in your downloads folder.

    Follow the Prompts: Click through the setup wizard (Next, Next, Install).

    Launch: Open Velozzy directly from your desktop.

# Development Setup

**If you want to run the source code or contribute to the project:**

    Python 3.10+

    PyQt6

    PyQt6-WebEngine

### Local Execution

    Clone the repository:
    Bash

    git clone https://github.com/keyzzilla/velozzy.git

    Install dependencies:
    Bash

    pip install -r requirements.txt

    Run the application:
    Bash

    python main.py
